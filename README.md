# Socket.IO TypeScript Template
A simple application in TypeScript with Node.js, that uses socket.io and sequelize


## Run instructions
Install nodejs
```
https://linuxize.com/post/how-to-install-node-js-on-centos-7/
```
Instal typescript
```sh
npm i -g typescript
```
Compile
```sh
npx tsc
```
Run 
```sh
node .
```
